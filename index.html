-- Server-side hitbox check local function checkHit(attacker, target) local dist = (attacker.HumanoidRootPart.Position - target.HumanoidRootPart.Position).Magnitude if dist <= 8 then target.Humanoid:TakeDamage(25) end end
Server-side hit validation
Distance check on server prevents exploits. Damage applied server-side only.
